# Fetal Movement Tracker

A React Native app built with Expo to track fetal movements. This project was created as part of an assignment.

---

## Installing the APK on Android

You can also install the pre-built APK directly on your Android device:

1. **Download the APK** from this link:
   [Download Fetal Movement Tracker APK](https://expo.dev/accounts/soham191003/projects/fetal-movement-tracker/builds/0a64f1c0-85c1-4a6c-95ec-499d862368bb)

2. **Install the APK**:

   * You may see a warning because the APK is from an unknown source.
   * Click the download link, then tap **Allow** if prompted to permit the installation.
   * Enable **Install from Unknown Sources** in your device settings if necessary.
   * Tap the downloaded APK to install the app.

> ⚠️ If an older version of the app is installed, uninstall it first to avoid conflicts.

## Running the App from Source

1. **Clone the repository**

```bash
git clone https://github.com/Soham-1910/FetalMovementTracker.git
cd FetalMovementTracker
```

2. **Install dependencies**

```bash
npm install
```

3. **Start the Expo development server**

```bash
npm start
```

4. **Run on Android device or emulator**

```bash
npm run android
```

> Make sure you have [Expo Go](https://expo.dev/client) installed on your device if using a physical device.


## Notes
